# OpenField

The dbt models in this project deliver the following deduplicated tables from Openfield:

- campaigns
- conversation_code
- conversations
- conversations_attempts_lookup
- customuser
- label
- people
- people_labels
- people_lookup
- question_dropdown_options
- question_responses
- questions
- scripts_question
- scripts
- turf

For more info on this OpenField data checkout the documentation on this site - [OpenField Export Pipeline Documentation](https://openfieldhelp.zendesk.com/hc/en-us/articles/38983933890068-OpenField-Data-Export-Pipeline)
